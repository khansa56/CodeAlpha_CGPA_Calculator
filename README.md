# CGPA Calculator

## 📌 Description
A C++ program that calculates Semester GPA and Overall CGPA.

## 🛠️ Features
- Input multiple courses with grades and credit hours
- Calculates weighted GPA
- Updates CGPA with previous academic record

## 🚀 How to Run
```bash
g++ cgpa_calculator.cpp -o cgpa_calculator
./cgpa_calculator
